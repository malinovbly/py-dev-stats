# django_proj
Проект на фреймворке Django по предмету "Технологии программирования на Python".

Автор: Новиков Артемий Алексеевич, АТ-16 (РИ-230930).

Ссылка: https://malinovbly.pythonanywhere.com

В веб-приложении "PyDevStatistics" можно ознакомиться с профессией "Python-программист", а также просмотреть статистику по этой профессии за промежуток времени, начиная с 2003 г., заканчивая 2024 г.


Страницы веб-приложения включают следующее:


1. Главная страница - описание профессии с тематическим изображением.


2. Общая статистика (статистика по всем вакансиям):
    
    - Динамика уровня зарплат по годам
    - Динамика количества вакансий по годам
    - Уровень зарплат по городам (в порядке убывания)
    - Доля вакансий по городам (в порядке убывания)
    - ТОП-20 навыков по годам


3. Востребованность (для профессии "Python-программист"):
    
    - Динамика уровня зарплат по годам
    - Динамика количества вакансий по годам


4. География (для профессии "Python-программист"):
    
    - Уровень зарплат по городам (в порядке убывания)
    - Доля вакансий по городам (в порядке убывания)


5. Навыки (для профессии "Python-программист"):
    
    - ТОП-20 навыков по годам


6. Последние вакансии - IT-вакансии соответствующие профессии "Python-программист" за последние 24 часа.
